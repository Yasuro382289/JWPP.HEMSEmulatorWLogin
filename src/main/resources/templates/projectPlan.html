プロジェクトの名称

エアコン自動調整システム

背景・目的など

近年電気代が高騰しており、電力消費を削減することが強く望まれている。一般家庭において電力消費を削減するためには、 消費電力の大きい機器、例えばエアコン、をこまめに制御するのがよい。近年のエアコンにはWi-Fiインタフェースが設けられ、 インターネット経由での制御が可能になっている。そこで今回、このエアコンの遠隔制御機能を前提とした エアコン自動調整システムをWebサービスとして作成しようと考えた。

これにより、家庭に設置された複数のエアコンの動作状況・各部屋の温度を監視し、 各部屋の室温が設定された範囲になるようにエアコンの動作状態を遠隔から自動制御することで、 家庭での総消費電力量を制御できるようになる。この他に、一か所から宅内のエアコンを制御できるので、 リモコンを探す手間を減らすこともできる。

利用する技術

  1. SpringBoot, SpringMVC
  2. HTML, Bootstrap, Thymeleaf
  3. Java
